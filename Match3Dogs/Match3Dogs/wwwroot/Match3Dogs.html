Match3Dogs
By Cooper Whitbread
Instructions:
    1. Press the text "Click Here To Play" to go to the menu screen.
    2. Pick what game modes you want by clicking the text of the desired game modes. The coloured circle indicates which mode has been selected.
    3. Press "Press Here to Begin!" to start the game.
    4. Start moving tiles any direction in a one tile radius to form match threes.
    5. Points are scored as follows: match 3 = 1p; match 4 = 2p; Match 5 = 3p; L (in any rotation) = 2p; T (in any rotation) = 2p.
    6. There is no check for possible matches but if the tiles need to reshuffled, press the button with an 'R' inside two arrows at the top right of screen.
    7. If at any point you want to pause, press the gear in the top right of screen.This is also how to exit the current game.
    8. To exit out of Match3Dogs, simply close the current internet explorer window.
